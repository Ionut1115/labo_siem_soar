# Azure Virtual Machine Infrastructure - Readme

This repository contains the Azure infrastructure configuration for deploying a Windows VM with IIS:

- **Virtual Machine (win-vm-iis-vm)**: The Windows VM running IIS.
- **Operating System Disk (myOsDisk)**: Disk for the OS and software.
- **Storage Account**: For VM data and temporary files.
- **Network Interface**: The VM's network connection.
- **Public IP Address**: External access to the web server.
- **Virtual Network**: Internal communication environment.
- **Network Security Group**: Controls inbound and outbound traffic for the VM.
